# Firefox Private Browsing Storage Tests

This example demonstrates how different web browser storage mechanisms (Local Storage, Session Storage, IndexedDB, and Cookies) behave when a web page is opened in Firefox's Private Browsing mode compared to normal mode. It highlights common failures or limitations encountered in private browsing, which is crucial for unit testing web applications to ensure they handle these scenarios gracefully.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in your Firefox browser (normal mode).
3. Observe the test results, then re-open `index.html` in a Firefox Private Window (Ctrl+Shift+P or Cmd+Shift+P) and compare the results.
